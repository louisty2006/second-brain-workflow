# ⚡ Quick Start — Get Running in 5 Minutes

You've built a second brain. Now let's use it.

## Step 1: Open Obsidian (1 min)

```bash
open -a Obsidian
```

Then:
- Click "Open vault as folder"
- Navigate to: `/Users/lautinyam/Documents/Louis-Brain`
- Click "Open"

You should see folders in the left sidebar.

## Step 2: Open Claude Code (1 min)

```bash
cd ~/Documents/Louis-Brain
claude
```

Claude Code opens with your vault as context.

## Step 3: First Command (30 seconds)

In Claude Code, type:
```
Start Work
```

Claude will:
- Read your identity (Self.md)
- Check your progress (session_handoff.md)
- Greet you and ask what you want to do today

## Step 4: Try /capture (30 seconds)

Type:
```
/capture I want to understand why webhooks need test URLs
```

Claude will save this to your Inbox.

## Step 5: End Session (30 seconds)

Type:
```
End Work
```

Claude will:
- Summarize what happened today
- Update session_handoff.md
- Prepare context for next session
