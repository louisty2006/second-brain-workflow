**VibeCode Reader** (macOS menu bar code explainer)

- API Key / Model 複製貼上：NSAlert → NSWindow、EditableTextField、Edit 選單；移除 Paste 按鈕
- Save 崩潰修復（`stopModal_` 不存在）→ 自訂 run loop 關閉視窗
- 啟動 crash 修復（`NSApp` 未就緒時呼叫 `_ensure_edit_menu`）→ 延遲至 app init
- Model 欄位貼上誤入 API Key → 焦點切換 + active field 判斷
- Accessibility 提示與檢測（`AXIsProcessTrusted`）；用戶加入 Homebrew Python.app 後快捷鍵可用

*晚段 — Bubble 穩定性*

- 繁體中文 + 英文翻譯均測試成功 ✅
