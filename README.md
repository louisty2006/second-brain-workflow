## Core Commands (Trigger Words)

### Phase 1 (Active Now)
- **Start Work** — Begin session with context
- **/capture [thought]** — Save raw idea to Inbox
- **/daily** — Create/update today's note
- **End Work** — Close session and update memory

### Phase 2 (After 1 Week)
- **/weekly** — Knowledge synthesis + connection finding
- **/connect [idea]** — Search vault for related notes

### Phase 3 (After 1 Month)
- iPhone voice capture → Obsidian
- Web Clipper for saving articles
- Advanced MCP automations

## The 3-Phase System Design

**Phase 1: MVP** (Week 1-2)  
→ Basic capture, daily notes, session continuity

**Phase 2: Strengthen** (Week 3-4)  
→ Weekly synthesis, connection finding, knowledge automation

**Phase 3: Advanced** (Month 2+)  
→ Voice capture, web research, AI-powered insights

You're here: **Building Phase 1 now**, with Phases 2 & 3 infrastructure ready.

## Token Efficiency Design

This system uses **Progressive Disclosure**:
- CLAUDE.md is minimal (导航 only)
- Each folder has its own instructions.md
- Claude reads on-demand, not everything at once

Result: ~700 tokens per session (not 5000+)

## Key Principles

1. **Capture > Organization** — Catch everything, organize later
2. **Connection > Collection** — Value is in relationships, not just data
3. **Ritual > Rules** — Daily/weekly habits beat perfect processes
4. **Both/And > Either/Or** — You're a creator + technologist, not one or the other
5. **Local > Cloud** — Your brain stays on your computer
