# Vault Index

This is a lightweight index for key decisions and cross-note pointers.

## How to use
- Add short decision entries with date and rationale.
- Link to related notes using wikilinks (for example, `[[05_Learning/topic-note]]`).
- Keep entries concise; detailed content belongs in project or learning notes.

## Decision Log
- YYYY-MM-DD: [Decision summary] -> [[related-note]]
